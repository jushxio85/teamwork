# 第十組- ```課課到，分分算```
我們的專題想呈現一個 校園小工具系統。

這個系統主要有兩個功能：
* 可以輸入課表，系統就會自動生成每週的課程提醒，幫助同學不會忘記上課。
* 可以輸入各科成績，系統會自動計算平均分數，讓同學隨時掌握學習狀況。
整體目標是讓同學更方便管理時間與成績，提高學習效率。
